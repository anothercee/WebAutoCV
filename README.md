WebAutoCV adalah alat daring (online) yang memungkinkan Anda membuat *Curriculum Vitae* (CV) atau resume yang profesional dan bersih secara cepat dan mudah. Hanya dengan memasukkan detail Anda ke dalam formulir interaktif, CV Anda akan dibuat secara *real-time* dan siap diunduh dalam format PDF.

## ğŸŒŸ Fitur Utama

* **Pembuatan CV Otomatis:** Input data Anda, dan CV akan terstruktur secara instan.
* **Pratinjau *Real-time*:** Lihat bagaimana CV Anda terlihat saat Anda mengetik, tanpa perlu *refresh*.
* **Ekspor ke PDF:** Unduh CV yang telah selesai dalam format PDF yang rapi dan siap cetak.

## ğŸš€ Instalasi dan Penggunaan (Pengembang)

Karena ini adalah aplikasi web statis (dijalankan di GitHub Pages), proses instalasinya sederhana.

### Prasyarat

* Git
* Node.js (Opsional, jika Anda menggunakan *bundler* seperti Webpack atau Vite)

### Langkah-Langkah

1.  **Kloning Repositori:**
    ```bash
    git clone [https://github.com/anothercee/WebAutoCV.git](https://github.com/anothercee/WebAutoCV.git)
    cd WebAutoCV
    ```
2.  **Instal Dependensi (Jika ada *package.json*):**
    ```bash
    npm install
    ```
3.  **Jalankan Secara Lokal (Jika ada *script* *start*):**
    ```bash
    # Contoh jika menggunakan script `start`
    npm run start
    # [Isi di sini: Ganti dengan perintah yang sesuai untuk menjalankan proyek lokal]
    ```
    Buka `http://localhost:[PORT_NUMBER]` di *browser* Anda.

### Cara Penggunaan (Sebagai Pengguna)

1.  Akses demo langsung di **[anothercee.github.io/WebAutoCV](https://anothercee.github.io/WebAutoCV/)**.
2.  Isi semua bagian formulir dengan informasi CV Anda (Data Diri, Pengalaman, Pendidikan, dll.).
3.  Perhatikan pratinjau yang diperbarui secara *real-time*.
4.  Klik tombol **"Unduh CV"** (atau sejenisnya) untuk mendapatkan file PDF.
